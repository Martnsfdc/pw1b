  • O programa definir automaticamente a margem da tag div

    • Display flex

  • O display: flex no CSS é usado para criar layouts flexíveis e organizar elementos dentro de um contêiner. Ele permite controlar o alinhamento, espaçamento e redimensionamento dos elementos automaticamente.

    • Funçao do .container no css

  • o CSS, um container é um elemento HTML que serve como caixa organizadora para outros elementos. Ele ajuda a controlar o tamanho, espaçamento, alinhamento e posicionamento dos itens dentro dele. Geralmente, um container pode ser uma div, section, main ou qualquer outro elemento HTML que agrupe conteúdos.
