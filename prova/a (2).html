Estudos

  • Começar a 1 pagina do site = Usar index.html
  • Tag h1 = Titulo dentro do site
  • Tag h2 = Sub titulo dentro do site
  • Tag h3 = Sub sub titulo dentro do site
  • Tag p = Representa um parágrafo
  • Tag li = esta junto com o ul e adiciona marcadores no site
  • Tag ol = abre uma lista (1,2,3.....)
  • Tag hr = adiciona uma lista horizontal no site
  • Tag head = Tag A = adiciona link
  • Tag div= adiciona uma divisao na tela, uma caixa, espaço
  • Tag tr = Linha na tabela
  • Tag table = começa uma tabela
  • Tag td= Celula da linha da tabela
  • Tag header= sao tags proprias para cabeçalho
  • Tag main= contém o conteúdo mais importante da página.
  • Tag aside = é usada para informações secundárias
  • Tag footer= rodape, é usada no final da página ou seção para informações adicionais.
